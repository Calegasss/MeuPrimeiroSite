# Meu Primeiro Site

Projeto que fiz em 2023 no SENAI para a disciplina de Sistemas Web.

Esse foi, na prática, meu primeiro projeto web maior. Eu fiz durante o curso Técnico em Assistente de Tecnologia da Informação, que cursei ao longo de 2023. A ideia era montar um site completo para uma marca fictícia de carros de luxo chamada `Unova`, com várias páginas, navegação entre seções e uma identidade visual própria.

## Sobre o projeto

O site foi feito com HTML, CSS, JavaScript e Bootstrap, com o conhecimento que eu tinha na época. Mesmo sendo um trabalho de curso, foi um projeto importante para mim porque foi uma das primeiras vezes em que montei algo mais completo e organizado na web.

Alguns detalhes do projeto:

- As imagens dos carros eu mesmo editei no Photoshop, colocando a identidade da marca em cima de super carros que já existiam.
- Os vídeos foram feitos usando IA, bem no começo dessa fase em que essas ferramentas estavam ficando mais populares.
- O trabalho foi desenvolvido no SENAI durante o ano de 2023.

## Estrutura atual

- `index.html`: entrada principal do site
- arquivos HTML principais na raiz
- `LAPRAS/` e `LUGIA/`: páginas internas de compra e contato
- `assets/images/`: imagens usadas no site
- `assets/videos/`: vídeos usados no site
- `lixo/`: arquivos antigos, duplicados ou que não estão em uso

## Paginas principais

- Home
- Home traduzida
- Models
- About Us
- Página de detalhes do modelo `LAPRAS GT`
- Página de detalhes do modelo `LUGIA`
- Fluxos de compra e contato por telefone

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap

## Como visualizar

Basta abrir o arquivo `index.html` no navegador.

## Contexto pessoal

Tenho um carinho especial por esse projeto porque ele representa uma fase bem importante do meu aprendizado. Foi uma das primeiras vezes em que consegui juntar código, identidade visual, edição de imagem e uma ideia mais completa de site em um trabalho só.
